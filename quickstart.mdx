---
title: "Quickstart"
description: "Get your docs running locally and make your first customization in minutes."
---

<Steps>
  <Step title="Clone your docs repo">
    Your docs repository was created during onboarding. Find the link in your [dashboard](https://dashboard.mintlify.com).

    Clone it to your local machine:

    ```bash
    git clone https://github.com/your-org/your-docs.git
    cd your-docs
    ```

    <Tip>If you haven't created a docs repo yet, use the Mintlify dashboard to initialize one from a template.</Tip>
  </Step>

  <Step title="Install the Mintlify CLI">
    Install the CLI globally using your preferred package manager:

    <CodeGroup>

    ```bash npm
    npm i -g mint
    ```

    ```bash yarn
    yarn global add mint
    ```

    ```bash pnpm
    pnpm add -g mint
    ```

    </CodeGroup>
  </Step>

  <Step title="Preview your docs locally">
    Run the local development server from inside your docs directory:

    ```bash
    mint dev
    ```

    Your docs are now live at `http://localhost:3000`. The preview updates automatically as you save changes.
  </Step>

  <Step title="Customize your site name and colors">
    Open `docs.json` in your editor. Update the `name` field and the `colors` object to match your brand:

    ```json
    {
      "name": "Your Project Name",
      "colors": {
        "primary": "#your-brand-color",
        "light": "#your-light-variant",
        "dark": "#your-dark-variant"
      }
    }
    ```

    Save the file and check `http://localhost:3000` to see your changes reflected immediately.

    <Tip>Try changing `primary` to a hex color like `#7C3AED` — the sidebar, buttons, and links all update instantly.</Tip>
  </Step>

  <Step title="Push to deploy">
    Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app) from your dashboard. It automatically deploys your docs on every push to your default branch.

    Commit and push your changes:

    ```bash
    git add docs.json
    git commit -m "Customize site name and colors"
    git push
    ```

    Your live docs site updates within moments of the push.
  </Step>
</Steps>
